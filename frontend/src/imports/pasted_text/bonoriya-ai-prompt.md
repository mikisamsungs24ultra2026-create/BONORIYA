# Prompt for Figma Make — Replace WhatsApp Support with Bonoriya AI

Replace the existing WhatsApp Support floating button displayed at the bottom-right corner of every page on the Bonoriya website with a new AI-powered virtual assistant called "Bonoriya AI."

## Objective

Bonoriya AI should become the primary customer support assistant across the entire Bonoriya website, providing instant, accurate, and user-friendly assistance to visitors, guests, partners, and prospective customers while maintaining Bonoriya's premium brand identity.

---

## UI Requirements

Replace the current WhatsApp floating widget with a modern AI chat widget.

The new widget should:

• Display the name "Bonoriya AI"
• Use an elegant AI/chat icon instead of the WhatsApp icon
• Match Bonoriya's premium branding and colour palette
• Be fixed at the bottom-right corner of every page
• Be fully responsive on desktop, tablet, and mobile
• Include subtle premium animations such as pulse, glow, hover lift, and smooth open/close transitions
• Have a clean, premium, and modern appearance

The chat window should include:

• Rounded corners
• Premium shadows
• Smooth animations
• Clean conversational interface
• Bonoriya brand colours
• Simple, uncluttered layout

• Complaints
• Legal or financial questions

• Any question outside the AI's knowledge base
